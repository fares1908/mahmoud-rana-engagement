# 🎵 Audio Instructions

Place your ambient romantic background music file here:

**File name required:** `ambient.mp3`

## Recommended Free Romantic Music Sources

1. **Free Music Archive** — https://freemusicarchive.org
   - Search "romantic ambient" or "engagement instrumental"

2. **Pixabay Music** — https://pixabay.com/music/
   - Search "romantic piano" or "soft piano ambient"

3. **Bensound** — https://www.bensound.com/royalty-free-music
   - Category: Romantic

## Format
- File: `ambient.mp3`
- Duration: 3–10 minutes (looped automatically)
- Style: Soft piano, strings, romantic ambient
- Volume: The website plays at full browser volume — use a soft track

## Note
If no audio file is present, the music button will still display correctly
and show a helpful tooltip. The rest of the website works perfectly without audio.
